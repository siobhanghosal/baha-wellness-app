# Wizard

## Scope

- Student onboarding
- Parent onboarding
- Consent setup
- Check-in sequencing

## Component Stack

- Stepper
- Cards
- Radio
- Checkbox
- Switch
- Buttons
- Dialogs

## Rules

- Each step contains one primary decision or action group.
- Progress visibility is mandatory.
- Safe backtracking is allowed unless policy or submission state would be invalidated.
