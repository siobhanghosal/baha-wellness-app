# BAHA Wellness Companion UI Prototype

Standalone Flutter UI prototype only. It does not connect to Supabase, FastAPI, Firebase, or any backend.

## Run

```bash
flutter pub get
flutter run
```

## What It Contains

- Student, Parent, Teacher, and BAHA Admin role experiences.
- Fake login/signup/onboarding flows.
- Dynamic student visual themes by gender and age group.
- Animated dashboards, cards, charts, dialogs, bottom sheets, and detail pages.
- Dummy data only.

## Design Intent

The UI is designed to feel like a near-production wellness app for stakeholder demos while remaining completely offline and backend-free.
